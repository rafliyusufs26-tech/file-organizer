# 📂 File Organizer
- My very awesome mini project in Python. Because it will organize the file really quickly without wasting my time creating a new folder and moving the file by myself. It is also really simple to use.

### ❓ How does the script work?
- Simple, when you run this script, it needs you to put the path folder in the input (if you are on Windows, make sure to use like this r"C:\Users\.." because it will give an error), and tada! Your file is organized in the folder with the same name as the extension file.
